# Data Jobs Market Analysis Dashboard (2024)

## Project Overview
This Power BI project provides a comprehensive analysis of the global data job market throughout 2024. The dashboard is designed to help job seekers, recruiters, and analysts understand salary benchmarks, geographic distribution, and specific requirements for various data-related roles.

### Data Source
The dataset consists of approximately **479,000 job postings** collected via **web scraping** from various platforms (LinkedIn, Indeed, BeBee, etc.) throughout the year **2024**.
